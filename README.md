# Lumière Skin Clinic Website 🌟

A premium, fully responsive website for a professional dermatology and skin care clinic.

## 🎨 Features

### ✅ **Fully Mobile Responsive**
- Optimized for all screen sizes (320px - 1920px+)
- Touch-friendly interface with 44px+ touch targets
- Smooth animations and transitions
- No horizontal overflow on any device

### 🏠 **Pages**
1. **Home** (`index.html`) - Hero section, services preview, testimonials, CTA
2. **About** (`about.html`) - Story, team, values, certificates with auto-slider
3. **Services** (`services.html`) - Complete treatment catalog with categories
4. **Gallery** (`gallery.html`) - Image gallery with lightbox and filters
5. **Contact** (`contact.html`) - Contact form, map, FAQ accordion

### 🎯 **Key Components**

#### Navigation
- Sticky navbar with blur effect
- Mobile drawer menu (slide from right)
- Topbar with contact info
- Smooth scroll navigation

#### Hero Sections
- Gradient backgrounds
- Floating badges with glass-morphism
- Image sliders (desktop & mobile)
- Professional slider controls
- Reduced padding for better spacing

#### Sliders & Carousels
- **Hero Image Slider**: Auto-play every 3 seconds
- **Certificate Slider** (About page): Auto-scroll on mobile
- **Core Values Slider** (Home page): Horizontal scroll on mobile
- Custom gold scrollbars
- Touch-friendly swipe

#### Interactive Elements
- Lightbox image viewer (Gallery)
- Filter buttons (Gallery)
- FAQ accordion (Contact)
- Animated stat counters (About)
- Hover effects on cards
- Smooth reveal animations

### 📱 **Mobile Optimizations**

#### Breakpoints
- **≤1024px**: Tablet layout, single column hero
- **≤768px**: Mobile sliders active, better spacing
- **≤640px**: Full mobile optimization
- **≤480px**: Extra compact for small screens

#### Mobile Features
- Hero section closer to navbar (20px padding)
- Horizontal scroll sliders for cards
- Full-width buttons and forms
- Stacked layouts
- Compact badges and tags
- Better touch targets (min 44px)
- Optimized typography
